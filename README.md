### How It Started 🚀

My coding journey began with the **4-week AVID project**—and let me tell you, it's been quite the adventure! 

I was already studying **JavaScript** when I decided to challenge myself further. I thought, "Why not try **Python** too?" 😅 Spoiler alert: **THIS IS REALLY HARD!** 

Between debugging, setting up my environment, and trying to figure out VS Code (that visual studio tool that's essential for building everything), I hit wall after wall. Especially opening VS Code? That alone felt like climbing a mountain! There were moments when I genuinely thought this challenge was way beyond what I could handle. I got stuck *a lot*, and I had to ask for help countless times—and that's okay. 

**But here's the thing: I didn't give up.** 💪

Now, looking back at what I've built with the Avid project, I'm genuinely proud. Not because it's perfect (spoiler: it's not), but because I *didn't quit* when things got tough. That's what growth looks like.
